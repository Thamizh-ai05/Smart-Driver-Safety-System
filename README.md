# AI Driven Alcohol Detection and Vehicle Safety System

The AI Driven Alcohol Detection and Vehicle Safety System is a smart driver monitoring system designed to improve road safety by detecting alcohol consumption, driver drowsiness, and abnormal health conditions.The system uses sensors, computer vision, and microcontrollers to monitor the driver's condition in real-time and take preventive safety actions such as speed control or vehicle shutdown. This project aims to reduce road accidents caused by drunk driving, fatigue, or medical emergencies.
---

## Features

- Alcohol detection using MQ-3 sensor
- Driver drowsiness detection using camera
- Blood pressure monitoring
- Vehicle speed control based on alcohol level
- Automatic vehicle shutdown for high alcohol levels
- Real-time monitoring dashboard

---

## System Architecture

The system integrates multiple hardware and software components:

1. Alcohol Sensor (MQ-3)
2. Camera for drowsiness detection
3. Microcontroller (Arduino / ESP8266)
4. Health monitoring sensors
5. Web dashboard for real-time monitoring

---

## Alcohol Detection Levels

Alcohol Level - Action 
Below 25% -- Normal driving allowed
25% - 50% -- Speed limited to 60 km/h
50% - 75%  -- Speed limited to 40 km/h
Above 75% -- Vehicle shutdown

## Methodology

1. Alcohol sensor detects BAC level from driver's breath.
2. Camera tracks eye movement and head position to detect drowsiness.
3. Health sensors monitor driver's vital parameters.
4. Microcontroller processes sensor data.
5. System applies safety actions like alert, speed control, or engine shutdown.

---

## Results

Testing results showed high accuracy in alcohol detection and fatigue monitoring.  
The system successfully limited vehicle speed or disabled ignition when unsafe driving conditions were detected.
